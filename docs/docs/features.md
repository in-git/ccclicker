# 功能特性

CCClicker 是一款功能强大的连点器工具，以下是它的主要特性：

## 核心功能

### ⚡ 极速响应
采用 Rust + Tauri 技术栈，提供原生应用般的流畅体验，点击延迟极低，确保精准执行。

### 🎯 精准控制
- 自定义点击间隔：支持毫秒级精度设置
- 自定义点击次数：可设置无限或固定次数
- 多种触发方式：支持快捷键、手动触发等多种方式

### 🔥 热键操作
- 全局热键控制：一键启动/停止连点器
- 无需切换窗口：在任何应用中都能快速控制
- 自定义热键：支持设置自己喜欢的快捷键

### 💾 数据持久
- 自动保存配置：所有设置自动保存到本地
- 快速恢复：下次启动自动恢复上次配置
- 多配置支持：可以保存和切换不同的配置方案

### 🎨 现代界面
- 简洁美观的 UI 设计
- 支持主题切换：明亮/暗黑模式
- 响应式布局：适配不同屏幕尺寸
- 流畅动画：提供舒适的用户体验

### 🔒 安全可靠
- 本地运行：所有数据都在本地处理
- 不上传数据：完全保护用户隐私
- 开源透明：代码开源，可审查安全性

## 使用场景

- **游戏辅助**：在游戏中实现自动点击功能
- **办公自动化**：重复性工作自动化处理
- **测试辅助**：软件测试中的点击测试
- **其他场景**：任何需要自动点击的场景

## 技术栈

- **前端**：Vue 3 + TypeScript + UnoCSS
- **后端**：Rust + Tauri
- **构建工具**：Vite

## 系统要求

- Windows 10 或更高版本
- 64 位操作系统
- 至少 100MB 可用磁盘空间
