# 快速开始

欢迎使用 CCClicker！本指南将帮助你快速上手使用这款强大的连点器工具。

## 安装

1. 前往 [GitHub Releases](https://github.com/in-git/ccclicker/releases) 页面
2. 下载最新版本的安装包
3. 运行安装程序，按照提示完成安装

## 首次使用

1. 启动 CCClicker 应用
2. 你会看到一个简洁的界面，包含以下主要部分：
   - **模式选择器**：选择连点模式
   - **配置面板**：设置点击参数
   - **开始按钮**：启动/停止连点器
   - **状态栏**：显示当前运行状态和统计信息

## 基本操作流程

1. **选择模式**：在模式选择器中选择你需要的连点模式
2. **配置参数**：在配置面板中设置点击间隔、次数等参数
3. **设置热键**：在设置中配置启动/停止的热键
4. **开始使用**：点击开始按钮或按下热键开始连点
5. **停止使用**：再次点击开始按钮或按下热键停止连点
