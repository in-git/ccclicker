# 基本使用

本章节介绍 CCClicker 的基本使用方法。

## 连点模式

CCClicker 支持多种连点模式，每种模式适用于不同的使用场景：

### 1. 单点模式
- 每次触发执行一次点击
- 适用于需要精确控制点击次数的场景
- 可以设置触发键和点击位置

### 2. 连点模式
- 持续执行点击，直到手动停止
- 适用于需要长时间自动点击的场景
- 可以设置点击间隔和次数限制

### 3. 间隔模式
- 按照设定的时间间隔执行点击
- 适用于定时操作的场景
- 可以精确控制每次点击的时间

## 配置参数

### 点击间隔
- 单位：毫秒
- 范围：1ms - 99999ms
- 建议：根据实际需求设置，避免过快导致系统卡顿

### 点击次数
- 可以设置为固定次数
- 也可以设置为无限次（0 表示无限）
- 实时显示已点击次数

### 触发键
- 支持键盘上的任意按键
- 建议使用不常用的按键，避免冲突
- 可以设置多个触发键

### 快捷键
- 启动/停止快捷键：全局控制连点器的启停
- 建议使用组合键，如 F6、F7 等
- 可以在设置中自定义

## 状态栏信息

状态栏会显示以下信息：
- **运行状态**：显示当前是否正在运行
- **已点击次数**：显示已经执行的点击次数
- **版本信息**：显示当前软件版本
- **反馈信息**：显示操作反馈和提示

## 使用技巧

1. **测试设置**：先设置较慢的点击间隔进行测试
2. **保存配置**：设置好参数后记得保存
3. **热键优先**：使用热键比手动点击更方便
4. **注意安全**：避免在重要操作中使用自动点击

## 常见问题

### Q: 点击太快怎么办？
A: 增加点击间隔的毫秒数，建议至少设置为 50ms 以上。

### Q: 如何停止连点器？
A: 点击停止按钮或按下设置的快捷键。

### Q: 配置会丢失吗？
A: 不会，所有配置会自动保存到本地。

### Q: 可以同时运行多个实例吗？
A: 目前不支持，建议只运行一个实例。
